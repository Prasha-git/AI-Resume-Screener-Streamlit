# AI-Resume-Screener-Streamlit

Overview

AI-based resume screening web app using Streamlit to analyze and filter candidate resumes.

🚀 Features

- User-friendly interface
- Real-time resume generation
- Dynamic input-based customization
- Clean and structured resume format

🛠️ Tech Stack

- Python
- Streamlit
- Pandas

📂 Project Structure

ai-resume-builder/
│── app.py
│── requirements.txt
│── README.md

▶️ How to Run

1. Clone the repository

git clone https://github.com/Prasha-git/ai-resume-builder-streamlit.git

2. Navigate to the project folder

cd ai-resume-builder-streamlit

3. Install dependencies

pip install -r requirements.txt

4. Run the application

streamlit run app.py
